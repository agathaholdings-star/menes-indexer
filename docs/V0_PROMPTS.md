# v0.app用 フロントエンドプロンプト集

## 概要

メンズエステ口コミサイト「メンエスインデクサ」のフロントエンドを生成するためのプロンプト集。
各ページごとにv0.appに投げるプロンプトを記載。

**技術スタック:**
- Next.js (App Router)
- Tailwind CSS
- ShadcnUI
- Lucide Icons
- Recharts（グラフ用）

**デザイン方針:**
- PCファースト + レスポンシブ
- 競合サイト（ME）に慣れたユーザーが迷わないUI
- アクセントカラー: #E11D48（ピンク/レッド系）
- 白背景ベース、情報量多め

## 1. トップページ


## 5. セラピスト詳細ページ（最重要）

```
メンズエステ口コミサイトのセラピスト詳細ページを作ってください。
これはサイトの核となるページで、レコメンド機能が最大の特徴です。

### ヘッダー
共通

### パンくずリスト
トップ > 東京 > 渋谷 > アロマモア > 田中まりあ

### セラピスト情報セクション（上部）
2カラムレイアウト:

**左カラム:**
- 写真ギャラリー（カルーセル、3枚程度）

**右カラム:**
- セラピスト名（大きく）+ 年齢
- 店舗名（リンク）
- スリーサイズ: T160 B86(D) W58 H85
- 一言コメント
- 出勤予定（今週のカレンダー形式）
- タイプタグ: #ギャル系 #グラマー #ドキドキ

### 分析データセクション（中部）
2カラムレイアウト:

**左: レーダーチャート**
Rechartsを使用。4軸:
- 会話（聞き上手↔話し上手）
- 距離感（まったり↔密着）
- 施術（リラックス↔ガッツリ）
- 性格（おっとり↔S気質）

**右: タイプ成分分析（円グラフ）**
「ギャル系 80%、アイドル系 20%」のような分布

### レコメンドセクション（重要！）
見出し: 「このセラピストが好きな人はこちらも」

横スクロール可能なカードリスト。各カード:
- 写真
- 名前、店舗名（別店舗も含む）
- タイプタグ
- 平均点
- 「同じギャル系で高評価」などの理由表示

### 口コミ一覧セクション（下部）

**口コミがブラー表示の場合（未ログイン or 権限なし）:**
- 口コミ本文にブラー（ぼかし）エフェクト
- 「口コミを投稿して24時間見放題」のCTAボタン（Sticky、画面下部固定）

**口コミが見れる場合:**
各口コミカード:
- 投稿日、点数
- タイプ・スタイル・サービスレベルのタグ
- パラメータ（小さいレーダーチャート or アイコン表示）
- 口コミ本文（Q1〜Q3の3つに分かれている）

### Sticky CTA（画面下部固定）
ログインしていない or 閲覧権限がない場合:
「口コミを投稿して無料で全て見る」ボタン（ピンク色、目立つ）

フレームワーク: Next.js (App Router), Tailwind CSS, ShadcnUI, Recharts, Lucide Icons
```

## 6. 口コミ投稿ウィザード（最重要）

```
メンズエステ口コミサイトの口コミ投稿ウィザードを作ってください。
Tinderライクな1問1答形式で、タップで簡単に進めるUIが特徴です。
モーダル or 専用ページどちらでも可。

### 全体の流れ
Step 0 → Step 1 → ... → Step 7 → 完了画面
進捗バーを上部に表示。戻るボタンで前のステップに戻れる。

---

### Step 0: 対象選択
- 店舗名: 入力欄 + オートコンプリート（DBからサジェスト）
- セラピスト名: 入力欄 + オートコンプリート or 「新しいセラピストを追加」ボタン
- 「次へ」ボタン

---

### Step 1: 見た目のタイプ（必須）
見出し: 「この子のタイプは？」
6つの大きなカードを2列×3行で表示。タップで1つ選択。

- 王道アイドル系 🎀（愛嬌抜群）
- 清楚なお姉さん系 👧（癒やし）
- イマドキなギャル系 👱‍♀️（ノリが良い）
- モデル・美女系 👸（高嶺の花）
- 親しみやすい妹系 👯‍♀️（元気）
- 妖艶な愛人系 💋（しっとり）

選択したら自動で次へ進む（または「次へ」ボタン）

---

### Step 2: スタイル（必須）
見出し: 「スタイルはどんな感じだった？」
4つのチップ or カードを横並び。タップで1つ選択。

- スレンダー
- 普通
- グラマー（巨乳）
- ぽっちゃり

---

### Step 3: サービス満足度（必須）
見出し: 「サービスはどうだった？」
4つのカードを2列×2行。アイコン付き。

- 健全 🧘（マッサージ重視）
- イチャイチャ 💕（恋人距離感）
- ドキドキ 💓（際どい施術あり）
- 過激 🔥（限界突破）

---

### Step 4: パラメータ評価（任意）
見出し: 「もう少し詳しく教えて」
4つのスライダー（1〜5）。それぞれ両極のラベルを表示。

- 会話: 聞き上手 ←→ 話し上手
- 距離感: まったり癒やし ←→ ドキドキ密着
- 施術: リラックス重視 ←→ ガッツリほぐし
- 性格: おっとりM気質 ←→ ちょっとS気質

「スキップ」ボタンもあり

---

### Step 5: アピールタグ（任意）
見出し: 「当てはまるタグを選んでね（複数可）」
チップ形式で並べる。タップでトグル。

**見た目系:**
#もちもち肌 #スレンダー #巨乳 #美脚 #黒髪 #派手髪

**性格・雰囲気系:**
#アニメ好き #お酒好き #彼氏感 #マッサージ自信あり #鼠蹊部

---

### Step 6: 点数（必須）
見出し: 「点数をつけるとしたら？」
大きな数字を中央に表示。スライダー or ボタンで10点刻み（0〜100）
または、10点ごとのボタンを並べる

---

### Step 7: 口コミ本文（必須）
見出し: 「最後に感想を教えて」
3つのテキストエリアに分割:

- Q1. 第一印象は？（写真とのギャップなど）: 50〜100文字
- Q2. 施術・接客で良かった点は？: 100〜150文字
- Q3. 気になった点・注意点は？: 50〜100文字

各テキストエリアの下に文字数カウンター表示

---

### 完了画面
紙吹雪アニメーション + 「投稿ありがとうございます！」
「24時間の閲覧権限が解放されました！🎉」（無料会員の場合）
「来月の割引ランクがUPしました！」（有料会員の場合）

「セラピストページに戻る」ボタン

フレームワーク: Next.js (App Router), Tailwind CSS, ShadcnUI, Framer Motion（アニメーション用）, Lucide Icons
```

## 使い方

1. 上記のプロンプトを1つずつv0.appにコピペ
2. 生成されたコードを確認・調整
3. 必要に応じて追加の指示を出す
4. 全ページ生成後、ルーティングを統合

**推奨順序:**
1. セラピスト詳細ページ（核となるページ）
2. 口コミ投稿ウィザード（核となる機能）
3. トップページ
4. 店舗詳細ページ
5. 検索結果ページ
